# Lumberaxe
🪓 Lumberaxe plugin for PocketMine-MP V4.*

# Features

```
✅️ Highly configurable
✅️ Good performance
✅️ Easy to use
✅️ Chop down trees extremly fast
✅️ Fast leaves decay
✅️ Support for custom recipe
✅️ Custom axe texture
✅️ Made with 💖
```
